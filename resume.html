Activities

The Egison Programming Language

http://www.egison.org

I have designed and implemented programming language Egison. Egison is the world's first programming language that realized non-linear pattern-matching with backtracking. We can directly represent pattern-matching against lists, multisets, sets, trees, graphs and any kind of data types.

Egison proposes a new paradigm pattern-matching-oriented. The combination of all of the following features enables intuitive powerful pattern-matching.

  • Modularization of the way of pattern-matching
  • Pattern-matching with backtracking
  • Non-linear patterns with lexical scoping

Egison is implemented in Haskell and distributed as a hackage package. Egison is getting popular and attracting a lot of people.

The source code is available on GitHub.
